flex layout 小记

2017年6月10日

Flex 布局主要思想是让容器有能力让其子元素能够改变其宽度、高度(甚至顺序)，以最佳方式(伸缩)填充可用空间，以防止它们溢出容器。

几个名词：

主轴：伸缩的子元素沿着主轴的方向排列布局，主轴的方向默认情况下是横着的，也有可能是竖着的，这个取决于 flex-direction 这个属性的值。主轴上的子元素的布局由 justify-content 这个属性来设置。
侧轴：与主轴相正交，侧轴上的子元素的布局由 align-items 这个属性来设置flex-wrap 用来定义伸缩项目夫元素 container 单行显示还是多行显示
